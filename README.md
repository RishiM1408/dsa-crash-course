# DSA Crash Course - SDE-2 Syllabus

This repository contains the code and curriculum for a high-intensity DSA crash course.

## Curriculum

1.  **Arrays & Hashing** (Prefix Sums & HashMaps) -> `src/main/java/com/dsa/arrays`
2.  **Two Pointers & Sliding Window**
3.  **Stack & Queue** (Monotonic Stack)
4.  **Linked List** (Fast/Slow Pointers)
5.  **Trees & Tries** (DFS/BFS Variants)
6.  **Heaps & Intervals**
7.  **Graphs** (Union-Find, Dijkstra, Topo Sort)
8.  **Dynamic Programming** (1D, 2D, Knapsack)
