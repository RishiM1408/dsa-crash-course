# DSA Crash Course - SDE-2 Syllabus

This repository contains the code and curriculum for a high-intensity DSA crash course.

## Curriculum & Solutions

1.  **Arrays & Hashing**: `Subarray Sum Equals K` (Prefix Sum)
    - [Source Code](src/main/java/com/dsa/arrays/SubarraySum.java)
2.  **Two Pointers & Sliding Window**: `Longest Repeating Character Replacement`
    - [Source Code](src/main/java/com/dsa/slidingwindow/LongestRepeatingCharacterReplacement.java)
3.  **Stack & Queue**: `Daily Temperatures` (Monotonic Stack)
    - [Source Code](src/main/java/com/dsa/stack/DailyTemperatures.java)
4.  **Linked List**: `Reorder List` (Fast/Slow Pointers)
    - [Source Code](src/main/java/com/dsa/linkedlist/ReorderList.java)
5.  **Trees & Tries**: `Lowest Common Ancestor` (BST Properties)
    - [Source Code](src/main/java/com/dsa/trees/LowestCommonAncestor.java)
6.  **Heaps & Intervals**: `Task Scheduler` (Priority Queue logic)
    - [Source Code](src/main/java/com/dsa/heaps/TaskScheduler.java)
7.  **Graphs**: `Number of Provinces` (Union-Find)
    - [Source Code](src/main/java/com/dsa/graphs/NumberOfProvinces.java)
8.  **Dynamic Programming**: `Coin Change` (1D DP)
    - [Source Code](src/main/java/com/dsa/dp/CoinChange.java)

## How to Run

Each file contains a `main` method with test cases. You can run them individually:

```bash
javac src/main/java/com/dsa/arrays/SubarraySum.java
java -cp src/main/java com.dsa.arrays.SubarraySum
```
